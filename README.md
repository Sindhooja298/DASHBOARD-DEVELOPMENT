Dashboard Development – Description

In this task, a dashboard is created to visualize student performance data using Power BI. The student-mat.csv dataset is used for this purpose, which contains information such as study time, age, gender, and final marks. The main aim of this task is to represent the data in a visual and interactive way for better understanding.

The dataset is first loaded into Power BI, and different charts are created to analyze the data. A bar chart is used to show the relationship between study time and average marks. A pie chart is used to display the distribution of students based on gender. A line chart is used to analyze how marks vary with age. A card is used to show the overall average marks of students.

In addition to these visuals, a slicer is added to filter the data based on specific categories like gender or school. This makes the dashboard interactive and allows users to explore the data easily.

Overall, this dashboard helps in identifying patterns and insights in student performance and makes data analysis more clear and effective.
